# L'élévation

A high-octane cyberpunk twin-stick shooter for Android, featuring fast-paced combat, deep upgrade systems, and a responsive neon-infused UI.

## 🚀 Key Features

### 🎮 Gameplay Mechanics
- **Twin-Stick Controls**: Intuitive dual virtual joysticks for precise movement and directional firing.
- **Dynamic Wave System**: Face increasingly difficult waves of cosmic horrors, culminating in challenging boss encounters.
- **Real-time Combat Engine**: A custom-built game engine utilizing `SurfaceView` for high-performance 2D rendering at 60 FPS.
- **Combat Systems**: Piercing projectiles, critical hits, area-of-effect explosions, and diverse enemy AI.

### ⚡ Upgrade Systems
- **In-Game Upgrades**: Choose between randomized mutations, weapons, and auras every time you clear a wave. Handled via a responsive, native Android UI overlay.
- **Permanent Progression**: Collect "Void Shards" during your runs to purchase permanent stat boosts and new abilities in the main menu.
- **Mutations & Auras**: Customize your build with passive effects like Static Fields, Plasma Shields, and Entropic Pulses.

### 📱 Advanced UI & UX
- **Responsive Design**: All game UI and menus are built to adapt perfectly to any screen size and aspect ratio.
- **Orientation Support**: Seamlessly toggle between **Portrait** and **Landscape** modes in-game via a dedicated orientation button.
- **Consolidated HUD**: A clean, horizontal dashboard at the top provides real-time feedback on Health, XP, Level, Current Wave, Score, and Kill count.
- **Cyberpunk Aesthetic**: High-contrast neon visuals, monospace typography, and smooth animations.

## 🛠 Technical Details
- **Architecture**: Modern Android development with **Kotlin**, **Dagger Hilt** for Dependency Injection, and **ViewBinding**.
- **Performance**: Optimized rendering loop with custom particle systems and frame-rate independent physics.
- **Persistence**: Game state and permanent upgrades managed via a secure Save Manager.

## 🕹 How to Play
1. **Move**: Use the left joystick to navigate the void.
2. **Shoot**: Use the right joystick to aim and fire, or let the auto-aim assist you.
3. **Ascend**: Collect XP shards and health packs from fallen enemies.
4. **Upgrade**: Select powerful augmentations after each wave to build your ultimate cosmic survivor.
5. **Survive**: Don't let the void consume you!
